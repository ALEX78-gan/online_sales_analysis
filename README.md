# online_sales_analysis

## Descriere
Proiect Python pentru gestionarea produselor și a unui coș de cumpărături.

## Clase

### Product
- name, price, quantity
- display_info()
- update_quantity()

### ProductManager
- add_product()
- display_products()
- total_value()
- remove_product()

### Cart
- add_to_cart()
- total_price()
- display_cart()

## Funcționalități
- gestionare produse
- calcul valoare inventar
- eliminare produse
- coș de cumpărături
